**Pré-requisitos:**
* Ter o compilador GCC instalado na máquina.
* Ter o Make instalado.

**Passo a passo rápido:**
1. Abra o terminal na raiz do projeto.
2. Compile o código executando: `make`
3. Para rodar todos os testes de uma vez, execute: `make test-all`
4. Para rodar um teste específico, use, por exemplo: `./lexer teste2.c`
5. A saída trará as colunas TIPO, LEXEMA e POSIÇÃO para sua análise.
